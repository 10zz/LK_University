Приложение личного кабинета студента и преподавателя университета.
Отображает информацию о расписании, мероприятиях, финансах и учебных ссылках для пользователя приложения.

Реализовано:
* Вход и создание аккаунта;
* Запоминание пользователя, если он уже входил ранее;
* Разграничение пользователей на 2 группы: Студенты и Преподаватели, для каждого изх них приложение выглядит по-разному;
* Отображение динамического списка мероприятий и учебных ссылок(с подгрузкой изображений из внешнего сервера) и финансов из внешней БД посредством библиотеки OkHttp;
* Отображение динамического списка расписания в соотетствии с выбором пользователем временного промежутка, группы и преподавателя из внешней БД посредством библиотеки OkHttp.

Находится в разработке:
* Метод загрузки и получения иконки профиля пользователя на внешний сервер;
* Метод отправки и обработки запроса о смене пароля;
* Метод заполнения списков преподавателя и групп для быстрого ввода пользователем.

Сборка и запуск:
1. Скачать и импортировать код в Android Studio(API 36.0);
2. Скачать и импортировать в Open Server Panel базу данных LKNNGASU;
3. Переместить директорию с PHP скриптами в директорию "domains" Open Server Panel;
4. В настройках Open Server выставить алиас с исходным доменом 10.0.2.2 и конечным доменом директории с PHP скриптами;
5. Запустить локальный сервер;
6. Запустить приложение.
